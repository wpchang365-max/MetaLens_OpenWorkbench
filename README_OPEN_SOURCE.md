# MetaLens Open Workbench

MetaLens Open Workbench is a brand-neutral open-source source package for metalens calculation, optimization, projection-lithography imaging, historical-data-assisted design, GPU/CPU resource management, and research export workflows.

This package intentionally removes institutional names, institutional logos, mottos, installation branding, special-edition licensing tools, and packaged EXE artifacts. The retained code focuses on computational methods, optimization methods, GUI workflows, data export, and reproducible research utilities.

## Run From Source

```powershell
py run_gpu_lens_modern.py
```

Optional acceleration libraries such as CuPy or PyOpenCL may be installed by downstream users according to their local GPU environment. CPU fallback remains available.
